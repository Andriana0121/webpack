npm run watch - vizualizam fișierele în timpul 'dezvoltării' proiectului
npm run build - compilam fișiere după dezvoltare
src - folder cu fișiere pentru dezvoltator cu cod, compilat în folderul dist
dist - fișiere compilate pentru proiecte
package.json - fisier cu dependenta a proiectului in care se contine descrierea proiectului si npm command
webpack.config.js - principalul fisier de configurare in care scrie ce fisiere din mapa src unde se compileaza, se conecteaza plugin
package-lock.json - descrierea tuturor plugins din mapa node_modules
bundle.js - fisier ce conecteaza codul pentru utilizare locala
